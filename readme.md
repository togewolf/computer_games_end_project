## How to use the keyword detection model:
0. make sure PyAudio is installed
1. Open terminal
2. Activate conda environment fulfilling requirements in voice_engine/requirements.txt
3. Run "python voice_bridge.py". Keyword detection is active now (see output in terminal)
4. Run the godot game to cast the spells
5. Speak the keywords:
	"igi": "fire", 
	"vova": "water",
	"rube": "nature",
	"pikat": "light",
	"zollzag": "target",
	"bumbo": "projectile",
	"noxo": "slow",
	"trodu": "fast",
	"pringo": "fastest"
